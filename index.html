I sei canali dell'IV Package

Ogni modo in cui un ospite
può contattarvi. Coperto.

WhatsApp Business

Messaggi in entrata e in uscita via Twilio WhatsApp Business API. Supporto immagini e PDF (conferme, mappe, menu). Il canale preferito dagli ospiti europei.

Twilio WhatsApp API

Voce (Chiamate)

Twilio Voice gestisce le chiamate in entrata: speech-to-text, risposta AI generata da Claude, text-to-speech in uscita. L'ospite chiama, l'AI risponde — nella sua lingua.

Twilio Voice

Email

Listener IMAP per la casella dell'hotel + relay SMTP per le risposte. Thread preservati via Message-ID. Il dominio dell'hotel rimane il mittente — brand consistency garantita.

SMTP / IMAP

SMS

Twilio Programmable Messaging per SMS in entrata e in uscita. Ideale per conferme di prenotazione, chiavi digitali e aggiornamenti rapidi — canale universale, nessuna app richiesta.

Twilio Messaging

Booking.com Messaggi

Tramite la Connectivity Partner API di Booking.com: ricezione e risposta ai messaggi degli ospiti direttamente nel thread della prenotazione. Zero ritardi, zero copiaincolla.

Connectivity Partner API

Recensioni

Risposta automatica alle recensioni Booking.com (e Google) tramite la stessa integrazione Connectivity Partner. L'AI risponde nel tono della struttura, mai con template generici.

Review API

Come funziona

Dal messaggio dell'ospite
alla risposta — in secondi.

01
L'ospite scrive
Su qualsiasi canale — WhatsApp, voce, SMS, email, Booking.com
02
Normalizzazione
Il canale adattatore converte il messaggio in uno schema unificato JSON
03
Contesto + AI
Il server MCP inietta tariffe, disponibilità e policy — Claude API genera la risposta
04
Verifica o auto-invio
Risposta automatica se entro la soglia di confidenza; altrimenti escalation allo staff
05
Risposta consegnata
De-normalizzata e inviata sul canale originale, nel thread corretto
